ezMP3

Members: Ryan Leiteritz

Application Overview

ezMP3 will be a program where users can quickly convert and download an online video link in .mp3 format. It is useful for archiving and backing up vulnerable sounds posted to video sharing sites (ie. YouTube).

Technology Overview

This program will be written in Python 3. A few libraries that ezMP3 may utilize are PyTube and youtube-dl. The goal for this program is durability. Since website APIs periodically change, ezMP3 will be written with resilience in mind. Due to a grey-area in legality, this program WILL NOT be hosted online in any public capacity. It will function as a standalone Python script that can be run on any operating system so long as Python 3 is installed. Any demo of ezMP3 presented by the creator will only be done so using videos he retains 100% exclusive copyright on.

Feature List

  1. Easy to use CLI
  2. Custom download location
  3. Produces 'true' .mp3 files, NOT AAC files with .mp3 tag. Files will be compatible with any program that supports .mp3 format.
  4. Queue multiple downloads at once.

Future Feature List

  • Support for more websites.
  • Option to directly download file as original .mp4.
  • Custom filename format.

Basic Technical Features

  • Command line interface
  • Processes a user's input to complete a task.
  • Extremely lightweight -- small application around 1 MB in size.

Advanced Technical Features

  • Utilization of Python libraries.
  • Scans website API for .mp4 files within a given URL. (Basic Web scraping)
  • Removes/splits video from the .mp4 file.
  • Compresses remaining audio using MPEG Layer III Audio Encoding (MP3).
